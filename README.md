# CSS Selectors Task

This project demonstrates the use of basic CSS selectors to style HTML elements.

## Features
- Connected external CSS file to HTML.
- Applied background color to the webpage.
- Styled multiple headings using a class selector.
- Used an ID selector to change the background color of a heading.
- Changed the color of a specific word in a heading.
- Styled paragraphs using element, class, and ID selectors.
- Customized text color and font size.
- Styled links without using inline CSS or applying class/id directly to the `<a>` tag.

## Files
- index.html
- style.css
- README.md

## How to Run
1. Download or clone the project.
2. Open `index.html` in any web browser.
3. The webpage will display the CSS selector styling as required.
